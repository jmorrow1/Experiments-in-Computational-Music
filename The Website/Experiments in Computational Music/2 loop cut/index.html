Loop / Cut (Experiments in Computational Music)

November 2015

Loop

Loops are the basis for many pieces of music, electronic and otherwise. The only real difference between loops in electronic music and loops in other forms of music is whether the loop is played by machine or musician. For our purposes here, we can treat these two cases the same. The act of looping audio is a powerful technique for musicians. It can even create the perception of music where otherwise we would only hear sound.

The thing about loops, of course, is that they are repetitive. It's this property that can make them compelling in some cases and grating in others.

Loop / Cut

We can introduce some variation into our repetitive loops when by manipulating them. One way to manipulate loops is to cut them. We can take a piece of audio, loop it (repeating it infinitely), cut it off at a finite point, and then loop that. We end up with a loop of audio that is locally the same as the loop we started with but globally different. Loop 2 is the result of applying that process to Loop 1. Press the waveforms to toggle playback on and off.

(Source of Loop 1: "Carnival" by David Kanaga)

You'll notice that the second loop behaves just like the first loop until it gets clipped off. The fact that it gets clipped off before it's allowed to fully repeat creates a new rhythmic pattern.

That was an example of the technique on raw audio. We can also use this loop/cut technique on musical phrases. For this, I've created a program that visualizes rhythms as grids of cells. Clicking on a cell changes it's state. Yellow cells are notes and white cells are rests. The slider controls the number of notes in the rhythm.


The band Meshuggah has a very particular way of using this technique to create music that is really rhythmically compelling. They'll start with an odd-time rhythmic phrase and repeat it. Then they'll loop it and cut it in such a way that it becomes a 4/4 phrase. They'll play this rhythm against a simple 4/4 rhythm on the snare and hi hat. The result is something that locally is polyrhythmic (playing an odd time against the 4/4 of the snare and hi hat), but globally is just a long rhythmic phrase in 4/4.

Higher Order Loops

Starting with a single loop, we can create a second loop using the idea of loop/cut. We can then use that second loop to create a third loop using loop/cut. We can continue on in this way an arbitrary number of times. We can think of this as a process of creating higher-order loops.
(Source of Loop 1: "Carnival" by David Kanaga)

We took a 2-measure loop and loop/cut it, turning it into a 5-measure loop. Then we took the 5-measure loop and loop/cut it, turning it into a 16-measure loop.

More Verbs

Loop/Cut is just one thing we can do to loops. If we wanted to, we could add more verbs to our vocabulary: splice, join, stack, flip, and so on. With the addition of more verbs the range of ideas we can express get larger. We can create more than just chains of looping and cutting. Now we can create families of loops of different shapes and sizes.
